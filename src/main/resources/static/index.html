This webapp provides a Opencast-like dummy GraphQL API. This way, it's easier to play with the API and everyone in the community can get familiar with GraphQL.

The actual API is available at /graphql. But to play with it, you probably want to use the included mini IDE GraphiQL, which is mounted here: /graphiql. On the right side, you can see the whole API schema. You can use ctrl+space to get auto-completions in the editor on the left.
